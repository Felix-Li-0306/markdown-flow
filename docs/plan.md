# Project Plan

## Project Name
iOS Markdown Editor

## One-line Summary
An open-source, mobile-first Markdown editor for iOS with separate source and preview modes.

## Goals
- Build a lightweight Markdown editor for iOS
- Support source editing and rendered preview
- Keep the interface simple and mobile-friendly
- Focus on local-first document editing

## Non-Goals
- Real-time Typora-style inline rendering
- Cloud sync in the first version
- Collaboration features
- Plugin system

## Target Users
- students
- note-takers
- developers who want a simple Markdown editor on iOS
- users who prefer writing in Markdown on mobile devices

## First Development Focus
- define the MVP
- design the app structure
- set up the iOS project
- implement document list, editor, and preview
