# MVP Definition

## Core Idea
The MVP is a simple iOS app that allows users to create, edit, preview, and save Markdown documents locally.

## MVP Features
- document list
- create new document
- source editing mode
- preview mode
- local save
- delete document

## Markdown Support for MVP
- headings
- paragraphs
- bold
- italic
- inline code
- code blocks
- unordered lists
- ordered lists
- blockquotes
- links

## Not Included in MVP
- cloud sync
- export to PDF
- collaborative editing
- themes
- advanced file management
- live inline rendering
- LaTeX and tables
